The Coffee Filter

Are you a coffee enthusiast? Share, rate, and discuss why coffee shops you've been to are hot or not.

The Coffee Filter is an application made with the Ruby on Rails framework using Test Driven Development. Built in a Swift and collaborative environment, The Coffee Filter uses Devise to authenticate users, Rails' intrinsic simple search along with simple forms, and CarrierWave to host images for user avatars. The project utilizes Git branching to increase efficiency and productivity.
